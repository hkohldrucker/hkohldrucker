What is this map?

The purpose of this map is to help visualize the distribution of the global economy. The GaWC index puts world cities into tiers based on their economic importance and connectivity. My hope is that creating an interactive map of this index would allow a user to analyze for themselves which regions of the world are most and least connected to the global economy. To learn more about GaWC, click here .

Methodology

How was this map made?

The first step in making this map was creating a GeoJSON file using the website geojson.io . I assigned each city a tier based on the GaWC 2020 rankings and also assigned each city a world region based on the World Bank's global regions (i.e. North America, Middle East and North Africa, etc.). After the GeoJSON file had been completed, I had to create a program which would essentially do two things: 1.) it needs to read the data from the GeoJSON file to create circle markers where each city is and 2.) it needs to allow the user to filter the cities based on the GaWC tier and the world region. The circle markers were assigned colors based on their GaWC ranking which was accomplished by creating a function called "getColor", which used else if statements to decide which color each circle marker should be. The filter was created essentially by asking Amazon Q to create a map which allowed a user to filter the circle markers using two different properties. Essentially two filters were created, one for GaWC ranking and one for region. Those filters were used in a separate function which ensures that only circle markers with each property checked show up on the map.
